## Controls

* `UP ARROW KEY`: Go forwards
  * Forwards is pretty much whichever direction the circle is facing  
    * `{s * dt * cos(θ), s * dt * sin(θ)}`
    * s = speed (in pixels)
    * dt = delta time
    * θ = angle in radians
* `LEFT ARROW KEY`: Rotate left
* `RIGHT ARROW KEY`: Rotate right
